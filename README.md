# LabSkill

Distill the research mind of elite academics. Generate reusable Skill files grounded in real literature.

## What it does

1. **Discover** — Find top researchers in any field using live web search
2. **Distill** — Pull papers from Semantic Scholar, bioRxiv, PubMed, plus auto-search public talks and interviews. Generate a SKILL.md with an Agentic Protocol derived from this researcher's epistemic standards
3. **Chat** — Ask questions. The PI searches current literature before answering — not from memory

## Agentic Protocol

Each distilled Skill includes an Agentic Protocol: a search strategy specific to that researcher's methodological standards. When Sanfey evaluates an experiment, he first checks for fMRI/behavioral convergence and replication status — because that's what his epistemological stance demands. Dana Small checks gut-brain mechanism literature and effect sizes first. The searches are different because the researchers are different.

## Data sources

- Semantic Scholar (papers, citations, abstracts)
- bioRxiv (preprints)
- PubMed (biomedical literature)
- OpenAlex (citation metrics)
- Web search (talks, interviews, lab websites)

## Disclaimer

LabSkill builds academic persona models from publicly available sources including published papers, public lectures, interviews, and lab websites. These models are research tools, not authoritative representations of any individual. They are not authorized by the researchers whose work is used, should not be cited as reflecting their actual views, and may contain inaccuracies. Use for research exploration only.
